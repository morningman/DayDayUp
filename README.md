# 🌟 Shawn 每日学习计划

一款简洁美观的每日学习计划管理工具，帮助孩子合理规划课后学习时间。通过可视化的活动模版和拖拽排序，轻松制定每日学习安排，并支持打印输出。

---

## ✨ 功能特性

- 📋 **活动模版库** — 内置多种学科活动模版（语文、数学、英语、练琴、体育等），也支持自定义新建模版
- 📅 **每日计划编排** — 点击模版即可快速将活动添加到今日计划中
- 🔄 **拖拽排序** — 通过拖拽调整活动顺序，灵活安排学习节奏
- ✏️ **活动编辑** — 可修改每项活动的时长和备注说明
- ⏰ **自动计算时间** — 根据设定的开始时间，自动计算每项活动的起止时间
- 🏷️ **分类筛选** — 按学科类别筛选活动模版，快速找到所需活动
- 🖨️ **打印输出** — 生成横版 A4 打印页面，包含完成打勾栏和备注栏，方便实际使用
- 💾 **数据持久化** — 自定义模版通过后端 API 保存到本地文件，不会丢失

## 📸 界面预览

应用由两栏组成：

| 左栏 - 活动模版库 | 右栏 - 今日计划 |
|---|---|
| 浏览和筛选所有活动模版 | 已选活动按时间顺序排列 |
| 点击模版添加到计划 | 支持拖拽排序、编辑、删除 |
| 支持新建自定义模版 | 一键打印当日计划表 |

## 🚀 快速开始

### 前提条件

- **Python 3.6+** — 用于运行后端服务
- 现代浏览器（Chrome / Firefox / Safari / Edge）

### 方式一：使用管理脚本（推荐）

```bash
# 启动服务
./server.sh start

# 查看状态
./server.sh status

# 停止服务
./server.sh stop

# 重启服务
./server.sh restart

# 查看日志
./server.sh log
```

### 方式二：直接运行

```bash
python3 server.py
```

启动后访问 **http://localhost:8080** 即可使用。

## 📖 使用指南

### 1. 选择日期和时间

在页面顶部设置 **日期** 和 **开始时间**（默认 19:00），系统会根据开始时间自动计算每项活动的起止时间段。

### 2. 添加活动到计划

在左侧「活动模版库」中：
- 使用顶部分类标签（全部 / 语文 / 数学 / 英语 / 练琴 / 休息 / 生活 / 体育 / 活动）进行筛选
- 点击活动模版卡片即可添加到右侧的「今日计划」中

### 3. 调整计划

在右侧「今日计划」中：
- 🔄 **拖拽排序** — 按住活动项上下拖拽，调整顺序
- ✏️ **编辑** — 点击编辑按钮，修改活动时长和备注
- 🗑️ **删除** — 点击删除按钮，移除单项活动
- 🗑️ **清空** — 点击顶部「清空」按钮，清空全部计划

### 4. 新建自定义模版

点击左侧「＋ 新建」按钮，可以创建自定义活动模版：
- 输入活动名称、默认时长、分类、图标（emoji）和说明
- 保存后会出现在模版库中，并持久化到服务器

### 5. 打印计划

点击「🖨️ 打印计划」按钮，将生成横版 A4 格式的打印页面，包含：
- 序号、时间段、分类、事项、完成打勾栏、备注栏
- 底部寄语鼓励

## 🔌 API 接口

后端提供以下 RESTful API：

| 方法 | 路径 | 说明 |
|---|---|---|
| `GET` | `/api/templates` | 获取所有自定义模版 |
| `POST` | `/api/templates` | 新增自定义模版 |
| `POST` | `/api/templates/delete` | 删除指定模版（传入 `{ "id": "..." }`） |
